* __Arrangement:__ The sequence of a given permutation's images of all its points 0, 1, …, n, also known as one-line notation. (Usually stored as truncated arrangement.)
* __Fixed point:__ A point that is mapped onto itself by a given permutation; a point that is not a moved point of that permutation.
* __Moved point:__ A point that is not mapped onto itself by a given permutation; a point that is not a fixed point of that permutation.
* __Perm, Permutation:__ A one-to-one mapping (bijection) of a (usually finite) set of points onto itself.
* __Point:__ An element of the set a permutation is defined on.
* __Support:__ The set of moved points of a given permutation.
* __Tarr:__ Short for truncated arrangement (see below).
* __Truncated arrangement, Tarr:__ The minimal prefix of a given permutation's arrangement containing that permutation's support.
